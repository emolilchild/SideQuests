# SideQuests
Sidequest is a webapp that helps people discover interesting things happening around them (small pop-ups, local events, hidden gems, places of interests). Users can explore explore a map, see nearby posts, and filter what they find by their interest.

# Features
1. Shows nearby discoveries ("Side Quests") on an interactive map
2. Filters content based on your interest (Art, Food, Music, Cars, Culture, etc)
3. See event info and short descriptions
4. Optional integration with external even sources for real-time events
5. Clean and easy-to-use map

# Project Scope
This version of "SideQuests" is a MVP designed to demonstrate the core idea (Discovering interesting nearby places through a simple map interface)
Full version includes:
- Custom user profiles and interest tracking
- Real-time notifications for nearby posts
- Social posting and sharing
- Image uploads and media integration
- Community discovery and friend connections
